# star-payroll
Payroll validation, bonus calculation, and formatting for import into Gusto.

# Usage
1. Validate time-clock manually (todo- include auto-validation)
2. Export reservations (check-ins), Instructor payroll, and employee payroll from MT
3. Include the files in a `/inputs`
4. Run `python main.py`
5. Outputs/final_gusto.csv can be imported into Gusto.
